

SFJBilling
====
SFJBilling - это простая бесплатная мини биллинг написанная на Laravel. SFJBilling могут пользоваться небольшие предприниматели оказывающую услуги.  

Основные функции 
----

- Учет абонентов. 
- Начисление услуг.
- Прием оплаты.
- Отчеты.
